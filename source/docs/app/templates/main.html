BMG UI-Kit dependencies

The BMG UI-Kit has following bower dependencies:

  • font-awesome
  • mindsmash-source-sans-pro

The BMG UI-Kit has following bower development dependencies:

  • jquery
  • angular
  • angular-bootstrap
  • angular-toastr
  • angularjs-slider
  • angular-ui-select
  • angular-ui-router
  • angular-sanitize
  • angular-xeditable
  • angular-aside
  • floatThead

To use some of these features you must include the required dependencies in your own bower.json.
