Configuration:

Property alues Explanation
ng-model Angular expression The model representing the typeahead's content
placeholder string Text displayed when the typeahead's text field is empty
oncommit function

Function to be called when the typeahead's value is committed

Parameters:

  • $data contains the entered value to be committed

Return value:

  • If the function returns a promise, the inline text field will display a spinning wheel until the promise is resolved or rejected.
  • If the promise gets resolved, a green success indicator is displayed.
  • If the promise gets rejected, a red error indicator is displayed. Additionally, the parameter with which the promise was rejected will be displayed as the error message underneath the text field.
  • If the function does not return a promise, the green success indicator is shown immediately.

items array, function The array of values to be used as the typeahead's possible values. Or a function for asynchronous calls (see also async).
async boolean For function calls only. If set you can provide a function for the attribute items.
tabindex number The index of this form field for tab navigation
disabled Angular expression If this expression evaluates to true, the input field will be disabled.
