Popover

Move the cursor over me


Popovers can be declared on an element using the regular UI Bootstrap Popover directive. Popovers can also specify custom templates containing arbitrary HTML. For detailed info on how to do this, consult the UI Bootstrap documentation. 

<span
    uib-popover="Content"
    popover-title="Title"
    popover-placement="auto top-left"
    popover-trigger="mouseenter">Move the cursor over me</span>
Dependency: angular-bootstrap
